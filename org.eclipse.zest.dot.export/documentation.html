Requirements: Eclipse 3.5 and GEF Zest visualization (Galileo)

Graphviz DOT as a DSL for Zest - M2: Graphviz export for Zest (Bug 277398)

  • A minimal sample setting up a Zest graph and exporting it to a DOT file and string can be found in src-test/org/eclipse/zest/DotExportSample.java, which can be run (as Java application) to see what the Zest input looks like; the result can be rendered using the Graphviz 'dot' command
  • A sample showing all that is currently supported in the Zest input can be found in src-test/org/eclipse/zest/dot/test_data/SampleGraph.java
  • See files in src-test/org/eclipse/zest/dot/test_data/ for usage of individual supported features (e.g. edge styles)
  • All samples contain a main method and can be run (as Java application) to see the Zest visualization
  • Run DotExportSuite in src-test (as JUnit or JUnit Plug-in test) to run all tests and transform all sample Zest graphs to DOT files
  • The generated files can be found in src-gen/ and can be rendered with Graphviz DOT to see the resulting visualization
